# smart-face-recognition-api

Final project, as part of Udemy course

To run the project 
<ul>
  <li>Clone the project</li>
  <li>Get your API key from Clarifai</li>
  <li>Run the following command to start the server</li> </ul>
  
  ```
  docker-compose up --build
  ``` 
  


To view a deployed version of the project, click [here](https://face-recognition1996.herokuapp.com/)
